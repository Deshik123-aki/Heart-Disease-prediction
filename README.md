# Heart Disease Prediction Using Random Forest Classifier

## Overview

Heart Disease Prediction Using Random Forest Classifier is a machine learning project developed using **Java**, **Apache Maven**, and the **Weka Machine Learning Library**. The primary objective of this project is to predict whether a patient is likely to have heart disease based on various medical attributes. The system uses the **Random Forest Classification Algorithm**, which is known for its high accuracy and reliability in classification problems.

This project demonstrates how machine learning can be integrated with Java to build an intelligent healthcare decision-support system. It provides practical experience in dataset processing, model training, evaluation, feature analysis, and prediction using real-world healthcare data.

---

## Features

* Predicts the likelihood of heart disease using machine learning.
* Implements the Random Forest Classification Algorithm.
* Developed using Java and Apache Maven.
* Utilizes the Weka Machine Learning Library.
* Loads and processes heart disease dataset from a CSV file.
* Trains and evaluates the prediction model.
* Performs feature importance analysis.
* Accepts user input for real-time prediction.
* Modular and well-structured Java implementation.

## Workflow

1. Load the Heart Disease Dataset.
2. Convert the dataset into Weka Instances.
3. Train the Random Forest Classifier.
4. Evaluate the trained model.
5. Analyze feature importance.
6. Accept patient medical parameters.
7. Predict whether heart disease is present.
8. Display the prediction result.

---

## Modules

* **Main.java** – Controls the execution of the application.
* **DataLoader.java** – Loads the dataset into Weka.
* **ModelTrainer.java** – Trains the Random Forest model.
* **Evaluator.java** – Evaluates model performance.
* **FeatureImportance.java** – Analyzes important medical attributes.
* **HeartDiseasePredictor.java** – Predicts heart disease based on user input.
